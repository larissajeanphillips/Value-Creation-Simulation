# Lessons Learned

This document captures important learnings, gotchas, and solutions to recurring problems encountered during development.

## How to Use

When you encounter a bug or issue that:
- Took significant time to solve
- Might happen again
- Has a non-obvious solution
- Would help future developers

Add it here with:
1. **Problem**: What went wrong
2. **Solution**: How you fixed it
3. **Prevention**: How to avoid it in the future
