# welcome

Getting started guide for new users.

This command will be available in chat with `/welcome`

When triggered, follow the `.cursor/rules/welcome.mdc` rule to provide the full welcome experience.

The welcome message includes:
1. **What you can create** - Inspirational examples showing the art of the possible
2. **How it works** - Simple explanation of Cursor and natural language coding
3. **Quick commands** - Essential slash commands to get started
4. **Tips for success** - How to get the best results
5. **Call to action** - Prompt to start building or use `/setup`

## Key Points to Emphasize

- **No coding experience required** - The AI writes the code
- **You can't break it** - Git saves everything, we can undo
- **Just describe what you want** - Natural language works
- **The only limit is what you can describe** - Inspirational closing
