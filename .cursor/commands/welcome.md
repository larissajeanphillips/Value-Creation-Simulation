# welcome

Getting started guide for new users.

This command will be available in chat with /welcome

When triggered, provide a comprehensive onboarding guide that includes:

1. **What this kit does** - Explain it's a starter template for creating React + TypeScript + Vite projects with Cursor AI support

2. **Quick Start (3 steps)**:
   - Step 1: Type `/setup` to launch the interactive project wizard
   - Step 2: Open the new project folder in Cursor
   - Step 3: Type `/launch` to start the dev server

3. **Available slash commands**:
   - `/welcome` - This guide
   - `/setup` - Launch the project setup wizard
   - `/commit` - Commit changes to git
   - `/commit-locally` - Commit without pushing
   - `/push` - Push to GitHub
   - `/launch` - Start dev server
   - `/restart` - Restart dev environment
   - `/review-code` - Code review checklist
   - `/add-lessons-learned` - Document issues

4. **Key files**:
   - `.cursorrules` - Project AI configuration
   - `ui/STYLE_GUIDE.md` - UI patterns
   - `ui/DESIGN_TOKENS.md` - Colors, fonts, spacing
   - `code-templates/` - Reusable code patterns
   - `docs-templates/` - Documentation templates

5. **Tech stack**: React 18, TypeScript, Vite, Tailwind CSS, Zustand

6. **Next steps**: Prompt user to type `/setup` to get started
