## Useful Phrases

These patterns work well when talking to the AI:

### Creating Things
- "Add a..." / "Create a..." / "Make a..."
- "Build a page that shows..."
- "I need a component for..."
- "Generate a form with fields for..."

### Changing Things
- "Change the... to..."
- "Make it more..." / "Make it less..."
- "Move the... to the..."
- "Swap the... and the..."
- "Update the text to say..."

### Styling
- "Make it look more professional"
- "Use a blue color scheme"
- "Add more whitespace"
- "Make the font bigger"
- "Add a shadow to..."

### Fixing Things
- "This looks wrong - it should..."
- "Fix this error: [paste the error]"
- "Why isn't this working?"
- "The button does nothing when I click it"

### Learning
- "Explain what this does"
- "What does [term] mean?"
- "Show me how to..."
- "Why did you do it this way?"

## Quick Troubleshooting

**App won't start?**
→ Try `/restart`. If that doesn't work, describe the error you see.

**Made a mistake?**
→ Press Cmd+Z (Mac) or Ctrl+Z (Windows) to undo. Or say "Undo the last change."

**Something broke?**
→ Say "Something broke after the last change. Can we go back?"

**Need to start over?**
→ Run `/setup` again to reconfigure your app.
